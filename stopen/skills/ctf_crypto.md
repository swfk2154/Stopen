# CTF 密码学 — Crypto 题目

## 目标
通过密码分析找到 flag。

## 方法

### 1. 识别加密类型
- **Base64**: 特征 A-Za-z0-9+/= 结尾
- **Hex**: 0-9a-f，长度偶数
- **ROT13**: 字母移位，特征可读
- **摩斯电码**: `.` `-` 和空格
- **栅栏密码**: 重新排列字母顺序
- **维吉尼亚密码**: 需要密钥的字母替换

### 2. 哈希识别
- MD5: 32 位 hex
- SHA1: 40 位 hex
- SHA256: 64 位 hex
- 彩虹表查询已知哈希值

### 3. 古典密码
- 凯撒密码：ROT3~ROT25 逐个尝试
- Atbash: A↔Z, B↔Y 对称替换
- 培根密码：5 位二进制组合

### 4. 现代密码
- RSA: 分解大素数、低指数攻击
- AES: 需要密钥，注意 IV
- XOR: 单字节 XOR 破解（频率分析）

### 5. 工具使用
- 使用 `crypto` 工具的对应操作
- 解码/解密后检查底部是否包含 flag 格式
