# 信息收集 (Reconnaissance)

## 目标
收集目标系统的信息，发现攻击面。

## 步骤

### 1. 端口扫描
- 使用 `port_scan` 工具扫描目标的开放端口
- 先扫常见端口（21,22,23,80,443,8080,8443,3306,6379,27017）
- 发现开放端口后再全端口扫描

### 2. 服务识别
- 分析端口扫描结果中的服务版本信息
- 对 Web 服务（80/443/8080/8443）：记录 Server 头、框架信息
- 寻找低版本/过期服务

### 3. 子域名枚举
- 使用 `subdomain_enum` 查找子域名
- 通过 crt.sh 和 DNS 枚举

### 4. 目录枚举
- 使用 `dir_brute` 扫描 Web 路径
- 重点关注：admin, api, .git, .env, backup, config, upload, 敏感文件

### 5. 技术栈识别
- 查看 HTML 注释、JS 文件、CSS 中的框架标识
- 使用 `http_request` 发送特定请求识别框架
